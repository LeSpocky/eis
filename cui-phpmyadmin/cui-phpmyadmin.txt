phpMyAdmin for eisfair-ng - MySQL-Administration mit phpMyAdmin
===============================================================

1. phpMyAdmin - Was ist das?
============================

Dieses Paket stellt das Datenbank-Administrationstool phpMyAdmin fuer MySQL
zur Verfuegung.



2. Die phpMyAdmin Menuepunkte
=============================

2.1 phpMyAdmin Hauptmenue
=========================

  1. View documentaion
     Zeigt diesen Text an.

  3. Configuration
     Mit diesem Menuepunkt wird die Konfiguration von phpMyAdmin vorgenommen.
     Eine genaue Erlaeuterung der Menuepunkte folgt unter
     "2.2 Edit configuration"

  4. Advanced configuration file handling
     Unter diesem Menuepunkt befinden sich die ueblichen Eis-Moeglichkeiten,
     die Konfiguration zu sichern, zu vergleichen etc. pp.

2.3 Migration von eisfair-1 bzw. eisfair-2

Soll eine vorhandene Konfiguration aus einer der vorherigen eisfair-Versionen
uebernommen werden, so kann die Datei 'config.inc.php' von dieser Installation
in den Ordner /etc/phpmyadmin/ kopiert werden, um diese Konfiguration zu
bearbeiten (Step 1, siehe oben) bzw. zu verwenden.
